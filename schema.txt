===================
Schema required for viewing daily assigned workout
===================

User:
------
id: integer, primary key
name: string
email: string
password: string
token:string
request_id:string
ath_profile_id: integer


Assignment:
---------
id: integer, primary key
date: datetime
workout_id: integer, foreign key 
total_miles_assigned: float  ?
total_time_assigned:datetime ?

Workout:
----------
id: integer, primary key
type:
pace: string   ?
workout_time: datetime  ?


A user has one assignment every day.
An assignment can be one of 9 workout types:
    1. Maintenance
An assigment has a pace and run_time that is specific to the user
